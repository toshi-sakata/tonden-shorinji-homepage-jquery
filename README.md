# 少林寺拳法 札幌屯田・丘珠スポーツ少年団 ホームページ（jQuery版）

既存の HTML + CSS + jQuery で構築していたホームページである。

React + TypeScript へリプレースする前の実装を比較用として公開している。

## React + TypeScript版

リプレース後のリポジトリはこちら。

https://github.com/toshi-sakata/tonden-shorinji-homepage

## 目的

本リポジトリは、既存サイトの構成や実装を確認できるようにするためのものである。

主な実装内容は以下。

- HTML / CSS による静的ページ構成
- jQuery によるアニメーション制御
- ハンバーガーメニュー制御
- スクロール時の表示制御
- レスポンシブ対応


## 技術構成

- HTML
- CSS
- jQuery
- JavaScript

## 備考

本リポジトリはリプレース前の比較用であり、今後の開発はReact + TypeScript版で行う。
