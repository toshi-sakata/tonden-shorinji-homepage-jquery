# 少林寺拳法 札幌屯田・丘珠スポーツ少年団 ホームページ（jQuery版）

少林寺拳法 札幌屯田・丘珠スポーツ少年団の公式ホームページです。

HTML、CSS、JavaScript、jQueryを使用して開発しています。

本リポジトリは、React版との比較および移行前の実装を参照する目的で公開しています。
今後の機能追加・改善はReact版で行います。

https://github.com/toshi-sakata/tonden-shorinji-homepage

## デモ

https://cf498068.cloudfree.jp/

## 主な機能

- レスポンシブデザイン
- ハンバーガーメニュー
- オープニングアニメーション
- スクロールフェードインアニメーション
- Googleマップによる練習場所表示

- HTML5
